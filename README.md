## Getting Started
### Installation
  1. Clone this repository
  ```
  git clone https://github.com/alan-dx/ignite-worldtrip.git
  cd ignite-worldtrip
  yarn install
  ```

  2. After install all dependencies
  ```
  yarn dev
  ```

  3. Open http://localhost:3000 on your browser
  
  4. Enjoy!
